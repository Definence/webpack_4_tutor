Run in a terminal:
```
npm init
npm install --save-dev webpack
npm i webpack-cli --save-dev
npm i html-webpack-plugin html-loader --save-dev
```

Add to package.json build script:
```
"scripts": {
  "build": "webpack"
}
```

Create: webpack.config.js

Create: ./scr/index.html

Run in a terminal:
```
npm run build
```
